GERADOR DE RELATÓRIOS NFSe – XML PARA EXCEL

Ferramenta gratuita para converter arquivos XML de NFSe
em planilhas Excel organizadas.

COMO USAR:
1. Execute o arquivo GeradorNFSe.exe
2. Selecione os XMLs ou uma pasta
3. Escolha onde salvar o Excel
4. Clique em Processar

INFORMAÇÕES IMPORTANTES:
- O software funciona 100% offline
- Nenhum dado é enviado ou coletado
- Não acessa portais ou sistemas externos

AVISO DO WINDOWS:
Na primeira execução, o Windows pode exibir um aviso.
Clique em “Mais informações” e depois “Executar assim mesmo”.
